¿Como controlar el flujo de mercancía y a su vez calcular el precio de venta de esta para la tienda mediante la implementación de un aplicativo web?

Objetivo General

Eliminar la falta de control en el inventario de la mercancía que tiene la tienda Texas Electronics mediante una aplicación web la cual se encargara de digitalizar toda la información.

Objetivos Específicos:

• Crear un modelo dentro de la aplicación web donde quede almacenada toda la información del inventario de los productos.

• Crear un Módulo donde se calcule el precio de venta de cada producto teniendo en cuenta el margen de rentabilidad por producto como también los impuestos que se generan por la venta de este.

• Crear un Módulo donde se muestre el flujo de movimientos de los productos con el fin de poder administrar todo el sistema.

Metodología

Realizar el diseño y la implementación de un aplicativo web que supla las necesidades tecnológicas que posee la tienda Texas Electronics.

Para esto se realizará como primera medida la arquitectura del sitio web.

Se trabajará con el patrón de diseño Modelo Vista Controlador en donde:

La Vista se desarrollará con los lenguajes HTML5 y CSS3 con el fin de exponer la información a los usuarios, este diseño se realizará intuitivo para facilitar el manejo de la información.

El Controlador se desarrollará con el lenguaje PHP quien se encargará de enviar y recibir las peticiones de información que el usuario final realice.

El Modelo se construirá sobre lenguaje SQL y la base de datos que usaremos para almacenar toda la información será la base de datos relacional (MySql).

Requerimientos Específicos

Funcionales


• Mantener toda la información de los productos actualizada.

• Los usuarios que usarán esta aplicación serán los vendedores y un administrador

• Los vendedores serán los encargados de generar la factura con el precio final de la mercancía incluyendo el impuesto y la utilidad de la empresa.

• El administrador será el encargado de ingresar la mercancía adquirida por los proveedores.


Técnicas


• El mantenimiento de la aplicación se realizará por medio de nosotros quienes diseñamos la aplicación

• La administración se hará por medio del rol del administrador.

• Para la codificación se usarán los lenguajes HTML5, CSS3, (frontend) y PHP (backend); como bases de datos se usará el lenguaje SQL para administrar la base de datos relacional.

• El contenido debe estar organizado de manera que sea intuitiva la interacción con la misma para adquirir la información que sea necesaria.

• El uso de esta aplicación es de manera constante y se hará de manera definitiva para solucionar el problema que se está presentando.

• El acceso lo tendrán los usuarios con los roles de vendedores y administrador respectivamente.

Finalidad

La finalidad de este proyecto es solucionar el problema que está presentando la tienda Texas Electronics la cual es la falta de control de inventario en los productos que ofrece al público, y a su vez aumentar la productividad y confianza en los clientes.

Cronograma

Recursos

El recurso humano son las personas que se encargarán del diseño y programación de la aplicación web. (Estudiantes de ingeniería de sistemas de la UNAD).

Los materiales son tanto el software como el hardware que cada una de las personas que se encargara del desarrollo de la aplicación, cabe nombrar que el software utilizado para dicho fin es software libre, por lo tanto, las maquinas a usar no necesitaran licencias para programar en los diferentes lenguajes.

No se contará con el recurso financiero ya que hace parte de un desarrollo que se hará con el fin de mostrar las habilidades técnicas para tal fin, a su vez como se mencionó anteriormente el software usado es libre. Por lo tanto, no se incurrirán en gastos para la elaboración del software.
